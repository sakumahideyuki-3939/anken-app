## 判定ロジック（日本語仕様・下書き）

### 入力（ユーザーが与えるもの）

- 日付（年月日）
- 時刻（任意／使わない可能性あり）
- 質問内容（テキスト）
- 選択項目（Yes / No / A・Bなど）

### 判定対象（何を見て、何を判断するか）

- 日付の干支・五行
- 質問内容のカテゴリ（恋愛／仕事など）
- Yes / No の選択結果

### 判定結果（最終的に何を返すか）
- 入力された日付が「どの要素（干支/五行/陰陽など）」に該当するか
- 質問内容が「どのカテゴリ（恋愛/仕事/金運/健康など）」に当たるか
- 選択項目（Yes / No / A / B など）によって、分岐ルールが変わるか
- 判定に使う基準が「固定ルール」か「日付で変動するルール」か
- 複数条件が同時に当てはまる場合の優先順位（どれを先に判定するか）
- 例外条件（判定不能/情報不足/入力ミス）の扱い
